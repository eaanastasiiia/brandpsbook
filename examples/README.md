# Примеры работ

Готовые примеры материалов в фирменном стиле ПСБ Спорт — референс "как это должно выглядеть в сборе".

| Файл | Описание |
|---|---|
| `82.png` | Титульный слайд презентации: «Семейный фестиваль ПСБ Спорт — Вместе, целая Россия!», фон — фирменный градиент, в углу — лого-вариация «Спортивная ПСБ Семья» |
| `81.png` | Слайд «Вводная информация» той же презентации: карточки что/когда/участники/где с фото стадиона |
| `Итоги_челленджа_основной_лист.pdf` | Инфографика-дашборд «Итоги глобального челленджа» — сводные цифры, тройка победителей, топ филиалов, график по месяцам |
| `Итоги_челленджа_топ-30.pdf` | Инфографика «Топ-30 победителей» того же челленджа — таблица лидеров + топ-3 по категориям активностей |
| `анна чичерова карточка 25.02.png` | Карточка амбассадора ПСБ Спорт — Анна Чичерова (олимпийская чемпионка), портрет + подпись, фирменный градиент-фон |
| `дмитрий труненков карточка 25.02.png` | Карточка амбассадора ПСБ Спорт — Дмитрий Труненков (олимпийский чемпион), тот же формат |
| `мск+ярик1.pdf` | Памятка участника «ЗаБег.РФ» (Москва + Ярославль) — текстовый документ на фирменном тёмно-синем/градиентном фоне |

Форматы: `.png` (слайды, карточки), `.pdf` (инфографика, памятки).
