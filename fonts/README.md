Заголовочный шрифт. Жирный, капс. Используется для заголовков (например, "ШРИФТ PSB SPORT").

## Формат файлов

Кладите файлы шрифтов (`.ttf` / `.otf`) в соответствующую подпапку, по одному файлу на начертание. Если полноценные файлы шрифтов грузить не будем — вместо этого положите сюда скриншоты текста, набранного каждым начертанием (с подписью, какое это начертание), чтобы стиль был понятен по картинке.
