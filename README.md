# Брендбук ПСБ Спорт

Материалы фирменного стиля отдела корпоративного спорта ПСБ — для использования как референс при создании простых дизайнов (презентации, посты, афиши, мерч и т.д.), в том числе с помощью Claude.

## О команде

**ПСБ Спорт** — отдел корпоративного спорта ПАО «Банк ПСБ». Отвечает за спортивные активности, мероприятия и коммуникации, связанные со спортом внутри банка: корпоративные челленджи (в приложении «В форме»), забеги и спортивные события для сотрудников и их семей (например, «Семейный фестиваль ПСБ Спорт», «ЗаБег.РФ»), работу с амбассадорами-олимпийскими чемпионами.

## Как использовать с Claude

Скидывайте Claude ссылку на нужную папку/файл (цвета, шрифты, паттерны, примеры) — и просите сделать дизайн в этом стиле. Основные вводные:

- **Шрифты**: см. [`fonts/README.md`](fonts/README.md)
- **Цвета**: см. [`colors/README.md`](colors/README.md)
- **Паттерны**: см. [`patterns/README.md`](patterns/README.md)
- **Логотип**: см. [`logo/README.md`](logo/README.md)
- **Примеры работ**: см. [`examples/README.md`](examples/README.md)
