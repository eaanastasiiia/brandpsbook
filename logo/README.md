# Логотип

Логотип ПСБ Спорт и его вариации.

| Файл | Описание |
|---|---|
| `лого псб спорт вертикаль цв.svg` | Основной вертикальный логотип, цветной (тёмно-синий щит, оранжевая звезда/стрелка, текст «ПСБ» на щите) |
| `лого псб спорт горизонт синий.png` | Горизонтальная версия: щит + текст «ПСБ СПОРТ» в тёмно-синем цвете |
| `лого_псб_спорт_щит_без_спорт.png` | Только щит (звезда/стрелка + «ПСБ»), без слова «СПОРТ» — компактный вариант |
| `лого_псб_оранж_бел_щит.png` | Оранжевый элемент щита (звезда/стрелка) отдельно, на прозрачном/белом фоне |
| `лого_псб_спорт_линия_оранж.png` | Горизонтальный текстовый логотип «ПСБ СПОРТ» одной строкой, оранжевый, со звездой вместо разделителя, без щита |
| `лого_псб_спорт_линия_белый.png` | Тот же текстовый логотип «ПСБ СПОРТ» в белом цвете — для использования на тёмном/цветном фоне |

## Форматы

Логотип хранится в `.svg` (векторный, предпочтителен для масштабирования) и `.png` (растровые версии с прозрачным фоном).

## Правила использования

Использовать как в примерах (см. [`examples/`](../examples/)). Чаще всего применяется **щит «ПСБ СПОРТ» сине-оранжевый, без слова «СПОРТ»** — файл `лого_псб_спорт_щит_без_спорт.png`.
