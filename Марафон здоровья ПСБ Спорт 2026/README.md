# Марафон здоровья ПСБ Спорт 2026

Материалы для конкретного мероприятия — «Марафон здоровья ПСБ Спорт 2026».

**Важно:** иконки, паттерны и шаблоны из этой папки используются только при работе над дизайном для Марафона здоровья 2026. Не открывай и не используй эту папку в остальных дизайнах — только если в задании явно попросили сделать материал именно для этого мероприятия.

Цвета и базовые паттерны те же, что и в общем брендбуке (см. [`../colors/README.md`](../colors/README.md) и [`../patterns/README.md`](../patterns/README.md)) — здесь только дополнительные, специфичные для этого события иконки/паттерны.
