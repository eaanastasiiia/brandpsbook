# Паттерны

Фирменные паттерны/текстуры ПСБ Спорт.

Кладите сюда файлы паттернов (`.png` / `.svg` / `.jpg`, желательно тайлящиеся/бесшовные, где это применимо).

## Иконки и мозаика — `Ресурс 2 3.png`

Набор фирменных пиктограмм спортивной тематики (секундомер, кубок, гиря, гантели, мяч, свисток, медаль, мишка, ракета, вертушка) в трёх цветах палитры — белый, оранжевый, сиреневый. Рядом — фирменный мозаичный паттерн (терраццо): рваные геометрические фрагменты в тёмно-синем/оранжевом/сиреневом, часть фрагментов — с диагональной штриховкой.

## Элементы для сборки паттернов — `pattern-parts/`

Отдельные векторные (экспортированные в PNG) элементы, из которых собираются готовые паттерны выше:

- `pattern-parts/icons/` — отдельные иконки (в т.ч. `dumbbell.png`, `kettlebell.png`) для сборки паттерна-россыпи
- `pattern-parts/mosaic-shards/` — отдельные фрагменты мозаики/терраццо для сборки паттерна в духе `Ресурс 2 3.png`

## Паттерн из иконок на градиенте — `Frame 2131328517.png`

Силуэты тех же спортивных иконок (мяч, вертушка, мишка, ракета и т.д.), полупрозрачно рассыпанные по фирменному градиенту (синий → сиреневый → персиковый). Используется как декоративный фон.

## Диагональный градиент-фон — `Frame 2131328518.png`

Чистый диагональный градиент (синий → сиреневый → персиковый) без дополнительных элементов, портретная ориентация — фон для карточек/сторис.

## Градиенты

Градиенты — один из видов фирменных паттернов ПСБ Спорт.

Самый базовый градиент используется как фон для объектов и текста, в качестве циклорамы, или как небольшой засвет в углу макета.

По общему правилу он создаётся в Cinema 4D — это позволяет получить нужную глубину цвета.

Для засвета можно сделать простой линейный градиент от фирменного синего (`#001F74`) до лилового (`#9F84FF`). Пропорции цветов произвольные, но синего должно быть больше.

### Вариант 1. Линейный градиент, циклорама — `gradient_1.png`

Розница, стр. 67. Простой линейный градиент от фирменного синего (`#001F74`) до лилового (`#9F84FF`).

### Вариант 2. Градиент в ракурсе — `gradient_2.png`

Сложный по форме трёхцветный градиент. Всегда используется как отдельный имиджевый элемент, не может быть подложкой для других объектов. Экспортируется из Cinema 4D или генерируется в нейросетях.

### Вариант 3. Градиент в ракурсе с задвоением — `gradient_3.png`

Самый яркий и заметный вариант градиента. Используется как отдельный имиджевый элемент, всегда является главным акцентом макета. Не может быть подложкой для объектов или текста. Экспортируется из Cinema 4D или генерируется в нейросетях.
