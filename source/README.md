# Исходники

Необработанные исходные материалы бренд-гайда (например, оригинальная презентация на 45 МБ), которые ещё предстоит разобрать на составляющие по папкам `logo/`, `colors/`, `fonts/`, `patterns/`, `examples/`.

Файлы отсюда не предназначены для прямой подачи Claude при создании дизайна — используйте разобранные материалы из других папок.
