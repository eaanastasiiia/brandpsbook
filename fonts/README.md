# Шрифты

Два фирменных шрифта ПСБ Спорт.

## PSB Sport — `PSB Sport/`

Заголовочный шрифт. Жирный, капс. Используется для заголовков (например, "ШРИФТ PSB SPORT").

- `PSB-Sport.otf` — единственное начертание
- `image_psbsport.png` — образец начертания

## Gilroy — `Gilroy/`

Текстовый шрифт, 10 насыщенностей от тонкого до чёрного, каждая — в прямом и курсивном начертании (20 файлов):

Thin, UltraLight, Light, Regular, Medium, SemiBold, Bold, ExtraBold, Heavy, Black — и Italic-версия каждой.

`image_gilroy.png` — образец всех начертаний.

## Формат файлов

Файлы шрифтов — `.otf`, по одному на начертание.
